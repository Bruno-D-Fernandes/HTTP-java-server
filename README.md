Este servidorzinho tem um pouco de tudo: é uma espécie de servidor RESTful (por mais que não tenha métodos como GET ou PUT por exemplo), 
com endpoints que simulam o funcionamento de ferramentas como o Apache.

O objetivo aqui é colocar em prática alguns conceitos que venho estudando sobre os fundamentos de redes e a relação entre hardware e web.
Apesar de não cobrir aspectos mais avançados como uso de threads ou integração direta com hardware, já quebra um bom galho para entender a base da comunicação entre navegador e servidor.
